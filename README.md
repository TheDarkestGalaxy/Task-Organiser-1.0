# Task Organiser

A simple, local task organiser to help you stay on top of what matters.

## Features

- **Create tasks** with a title and optional description
- **Set importance** — Low, Medium, High, or Critical
- **Bookmark tasks** to pin the ones that need your attention
- **Filter** by importance or bookmarked status
- **Auto-saves** to your browser (localStorage)

## How to use

1. Open `index.html` in your web browser (double-click the file)
2. Type a task title, pick an importance level, and click **Add Task**
3. Click the star (☆) to bookmark a task — bookmarked tasks appear at the top
4. Use the filter pills to view All, Bookmarked, or tasks by importance

No installation required.
