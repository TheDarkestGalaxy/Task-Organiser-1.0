# Task Organiser

A simple, local task organiser to help you stay on top of what matters.

## Features

- **Create tasks** with a title, optional description, due date, and category
- **Set importance** — Low, Medium, High, or Critical
- **Bookmark tasks** to pin the ones that need your attention
- **Mark complete** with checkboxes — completed tasks move to the bottom
- **Due dates** — overdue and due-today tasks are highlighted
- **Categories** — organise tasks (e.g. Work, Personal) with filter pills
- **Filter** by status, importance, or category
- **Auto-saves** to your browser (localStorage)

## How to use

1. Open `index.html` in your web browser (double-click the file)
2. Type a task title, pick importance, optionally set a due date and category, then click **Add Task**
3. Check the box to mark a task complete
4. Click the star (☆) to bookmark a task
5. Use the filter pills to view Active, Completed, Bookmarked, by importance, or by category

No installation required.
